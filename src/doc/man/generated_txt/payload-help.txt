PAYLOAD-HELP(1)

NAME
       payload-help - Get help for a Payload command

SYNOPSIS
       payload help [subcommand]

DESCRIPTION
       Prints a help message for the given command.

EXAMPLES
       1. Get help for a command:

              payload help build

       2. Help is also available with the --help flag:

              payload build --help

SEE ALSO
       payload(1)
